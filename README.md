# vite-10syunen

Vite + vanilla JavaScript + vanilla CSS + three.js + ammo.js で作ったミニゲームです。

- 500px の箱の中に、180px 相当の小石を 10 個落とします。
- 全小石が床に接して静止状態を一定時間維持するとクリアです。

## セットアップ

```bash
npm install
```

## 開発起動

```bash
npm run dev
```

## 本番ビルド

```bash
npm run build
```

## 操作

- `リスタート` ボタンで小石の配置をリセットできます。
